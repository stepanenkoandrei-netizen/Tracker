# Tracker - Финансовый Telegram бот

Бот для учета доходов и расходов с интеграцией Google Sheets.

## Функции
- 📊 Быстрый и пошаговый ввод операций
- 📈 Графики и отчеты
- 📋 История операций
- 🗑 Удаление операций
- ⚙️ Админ-панель для управления категориями

## Деплой на Render
1. Подключите GitHub репозиторий
2. Добавьте переменные окружения:
   - BOT_TOKEN
   - ADMIN_ID
   - GOOGLE_SHEET_KEY
   - GOOGLE_CREDS (содержимое cred.json)
3. Нажмите Deploy
